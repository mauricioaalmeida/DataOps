## üõ†Ô∏è Arquitetura

### Componentes
- **Rede**: `ds-network` conecta todos os servi√ßos.
- **Persist√™ncia**:
  - Volume unificado: `./data` (e.g., `./data/postgres`, `./data/airflow`).

  - Kafka UI: `8082`
  - Jupyter: `8888`
  - MinIO: `9000` (API), `9001` (Console)
